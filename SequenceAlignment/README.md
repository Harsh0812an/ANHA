# Sequence Alignment

The **Sequence Alignment** module is one of the core components of the ANHA (Advanced Nucleotide Health Analyzer) project. It compares two DNA sequences and determines the optimal global alignment using a dynamic programming approach.

This alignment provides a standardized representation of the sequences, making them suitable for downstream comparative analyses.

---

## Features

- Accepts two DNA sequences as input
- Performs global sequence alignment
- Uses a scoring-based algorithm to determine the optimal alignment
- Displays the aligned DNA sequences
- Calculates the final alignment score
- Forms the basis for comparative DNA analysis

---

## Scoring System

| Operation | Score |
|-----------|------:|
| Match | +1 |
| Mismatch | -1 |
| Gap | -2 |

The alignment with the highest possible score is selected as the optimal alignment.

In future versions, the Mutation Detector will be separated into its own module while continuing to use the Sequence Alignment module as its dependency.
